# 🤝 מדריך תרומה לפרויקט Rafael

תודה על התעניינותכם בתרומה לפרויקט! אנו מעריכים כל תרומה, קטנה כגדולה.

## 📋 איך לתרום

### דיווח על באגים

אם מצאתם באג:

1. בדקו אם הבאג כבר דווח ב-[Issues](https://github.com/AnLoMinus/Rafael/issues)
2. אם לא, פתחו issue חדש עם:
   - תיאור ברור של הבאג
   - צעדים לשחזור הבעיה
   - התנהגות צפויה מול התנהגות בפועל
   - דפדפן ומערכת הפעלה
   - צילומי מסך (אם רלוונטי)

### הצעת שיפורים

להצעת תכונה חדשה:

1. פתחו issue עם תיאור מפורט של התכונה
2. הסבירו למה התכונה תהיה שימושית
   - האם היא מתאימה לפרויקט?
   - איך היא תתאים לשאר המאגר?

### תרומה בקוד

1. **Fork** את הפרויקט
2. צרו **branch** חדש (`git checkout -b feature/AmazingFeature`)
3. בצעו את השינויים שלכם
4. **Commit** את השינויים (`git commit -m 'Add some AmazingFeature'`)
5. **Push** ל-branch (`git push origin feature/AmazingFeature`)
6. פתחו **Pull Request**

## 📝 כללי קוד

### סגנון קוד

- השתמשו ב-2 רווחים (לא טאבים) לזיהוי
- שמות משתנים ופונקציות באנגלית
- הערות בקוד בעברית (אם נדרש)
- שמרו על קוד נקי וקריא

### HTML

- השתמשו בסמנטיקה נכונה
- שמרו על מבנה נגיש
- הוסיפו alt text לתמונות

### CSS

- השתמשו ב-CSS Variables
- שמרו על עקביות בשמות
- השתמשו ב-Flexbox/Grid באופן מושכל

### JavaScript

- השתמשו ב-ES6+ syntax
- שמרו על פונקציות קצרות וממוקדות
- הוסיפו הערות במקומות מורכבים
