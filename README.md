# 💞 Rafael

פרופיל שידוך מקיף ואותנטי לרפאל איסקוב.

## 🌟 אודות

Rafael הוא מאגר דיגיטלי מקיף להצגת פרופיל שידוך מלא ומפורט. המאגר כולל את כל ההיבטים הרלוונטיים: פרטים אישיים, רקע רוחני, אופי ותכונות, עיסוק וייעוד, חזון חיים, ומה מחפש בזוגיות.

## 📋 תוכן המאגר

המאגר כולל את הסעיפים הבאים:

- 🪪 **פרטים אישיים** - שם, גיל, מקום מגורים, שפות
- 🌿 **רקע רוחני** - השקפת עולם, קהילה, רב מלווה
- 🧬 **אופי ותכונות** - תכונות דומיננטיות, סגנון תקשורת
- ⚙️ **עיסוק וייעוד** - תחום עיסוק, מטרות קריירה
- 💫 **חזון חיים** - ערכים, חזון בית, דמות זוגיות
- ❤️ **מה מחפש בזוגיות** - תכונות חשובות, רמה דתית
- 🕊 **תחביבים ואורח חיים** - פנאי, תחומי עניין, מוזיקה
- 🔮 **תמונה אנרגטית** - אנרגיה כללית, מה מאיר
- 📸 **מדיה ותמונות** - תמונות עדכניות ומכבדות
- 💞 **טבלת התאמה זוגית** - דירוג עצמי במגוון פרמטרים

## 🚀 התקנה ושימוש

### דרישות

- דפדפן מודרני (Chrome, Firefox, Safari, Edge)
- אין צורך בהתקנות נוספות

### שימוש מקומי

1. שכפול המאגר:

```bash
git clone https://github.com/AnLoMinus/Rafael.git
cd Rafael
```

2. פתיחת הקובץ `index.html` בדפדפן

### שימוש ב-GitHub Pages

האתר נגיש דרך: **<https://AnLoMinus.github.io/Rafael/>**
